A robust, scalable, and user-friendly E-Learning management system built to streamline the online education process. This project focuses on delivering a seamless experience for both instructors to manage their content and students to enhance their skills.

---

## 🚀 Technologies Used

* **Framework:** .NET Core / ASP.NET Web API (C#)
* **Frontend:** Blazor, HTML5, CSS3, JavaScript, and Bootstrap
* **Database:** SQL Server (Relational Database Design)
* **Architecture:** N-Tier Architecture / Clean Architecture
* **Design Patterns:** Repository Pattern & Unit of Work
* **Security:** Identity Framework for Authentication & Role-based Authorization

---

## 🌟 Key Features

* **User Management:** Secure Sign-up/Login with multi-role support (Admin, Instructor, Student).
* **Course Management:** Full CRUD operations for courses, lessons, and categories.
* **Enrollment System:** Students can browse and enroll in various courses.
* **Progress Tracking:** Ability to monitor student progress throughout the curriculum.
* **Interactive Dashboard:** Comprehensive stats and management tools for administrators.
